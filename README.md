## Getting Started with Chapa

First, run npm i or npm install to install all necessary modules
Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx` also mess with the global css with the colors. The page auto-updates as you edit the file.
